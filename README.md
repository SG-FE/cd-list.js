# CdList

cdlist 提供了 web 端列表应用的解决方案，通过配置插件可以实现分页、搜索、筛选、排序等功能；所有的操作通过 history API 记录，可以实现前进、后退、url 定位选中状态等。

### 查看示例

点击，[CdList Gitpage](https://sg-fe.github.io/cdlist/) 查看各种插件的使用方法
